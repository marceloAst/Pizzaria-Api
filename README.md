#🍕 Pizzaria API

API RESTful desenvolvida para gerenciamento de uma pizzaria com atendimento presencial (sem delivery). O projeto foi desenvolvido como Trabalho Prático da disciplina de Arquitetura de Software, aplicando conceitos de Arquitetura Vertical Slice, Orientação a Objetos, Injeção de Dependência e modelagem de banco de dados relacional utilizando PostgreSQL.

##📚 Objetivo

A aplicação permite gerenciar os principais processos de uma pizzaria, incluindo:

Cadastro de clientes;
Cadastro de categorias de produtos;
Cadastro de produtos (pizzas, bebidas e sobremesas);
Cadastro de mesas;
Controle de pedidos;
Registro de pagamentos.
##🛠 Tecnologias Utilizadas
Node.js
Fastify
PostgreSQL
pg
dotenv
Swagger (OpenAPI)
JavaScript (ES6)
